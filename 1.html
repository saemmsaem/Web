HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers recieve html documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a wev page~ document.

Html elements are the buliding blocks of html pages. With html constructs, imageds and other objects, such as interactive forms, may be embedded into the rendered page.
